Công nghệ sử dụng

- ASP.NET Web Forms (.aspx)
- C#
- SQL Server
- ADO.NET
- JavaScript (Inline Alert)
- HTML + CSS

Frontend: Trangsanpham.aspx
Backend: Trangsanpham.cs

Cách chạy dự án:

- Clone hoặc tải dự án
- Mở bằng Visual Studio
- Cấu hình chuỗi kết nối database trong Web.config
- Chạy dự án
